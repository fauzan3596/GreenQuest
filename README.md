# Panduan Fitur GreenQuest AI 🌱

Selamat datang di **Panduan Pengguna GreenQuest AI**. Dokumen ini menjelaskan secara mendalam berbagai fitur yang tersedia dalam aplikasi untuk membantu Anda memahami cara kerja dan cara memaksimalkan penggunaan platform ini.

Akses aplikasi secara online melalui tautan berikut:
🚀 **[GreenQuest AI Web App](https://greenquest-swzj6evuj6nz2kmtgeewvx.streamlit.app/)**

---

### 📊 1. Dashboard Interaktif
Dashboard adalah pusat informasi emisi Anda. Di sini Anda dapat melihat:
*   **Metrik Total Emisi**: Ringkasan akumulasi seluruh emisi karbon Anda dalam satuan kg CO2.
*   **Grafik Distribusi (Pie Chart)**: Visualisasi yang menunjukkan kategori aktivitas mana yang paling banyak menyumbang emisi. Ini membantu Anda mengidentifikasi "pelaku utama" jejak karbon Anda.
*   **AI Analysis (EarthBot)**: Fitur unggulan yang menganalisis data Anda secara otomatis dan memberikan tips pengurangan emisi yang disesuaikan dengan profil data Anda.

### ➕ 2. Pencatatan Aktivitas (Batch Input)
Anda dapat mencatat berbagai aktivitas harian secara kolektif sebelum menyimpannya secara permanen:
*   **Daftar Sementara**: Masukkan aktivitas satu per satu dan klik **"Tambah ke Daftar Sementara"**. Anda akan melihat notifikasi konfirmasi di pojok kanan bawah.
*   **Review Data**: Tabel sementara akan muncul untuk membantu Anda meninjau aktivitas yang telah dimasukkan.
*   **Submit Masal**: Klik **"Submit Semua Data"** untuk memindahkan seluruh daftar ke riwayat permanen dan memperbarui dashboard.
*   **Batal**: Gunakan tombol **"Kosongkan Daftar"** jika ingin menghapus seluruh daftar sementara.

Aktivitas yang didukung:
*   **Listrik (Electricity)**: Masukkan jumlah penggunaan dalam kWh.
*   **Bensin (Petrol)**: Masukkan konsumsi bahan bakar kendaraan dalam liter.
*   **Daging (Meat)**: Masukkan berat konsumsi daging (sapi/domba) dalam kg.
*   **Transportasi Umum**: Masukkan jarak tempuh perjalanan dalam km.
*   **Penerbangan (Flight)**: Catat jumlah penerbangan jarak pendek yang dilakukan.

### 💬 3. Chat with EarthBot AI
Fitur ini adalah asisten pribadi Anda untuk gaya hidup berkelanjutan. EarthBot bukan sekadar chatbot biasa karena:
*   **Konteks Pintar**: EarthBot mengetahui data emisi Anda saat ini, sehingga ia bisa menjawab pertanyaan seperti *"Bagaimana cara mengurangi emisi saya yang didominasi oleh penggunaan listrik?"*.
*   **Edukasi**: Anda bisa bertanya tentang konsep perubahan iklim, cara daur ulang, atau alternatif barang ramah lingkungan.
*   **Motivasi**: Memberikan dorongan positif untuk terus melakukan perubahan kecil demi masa depan bumi.

### ⚙️ 4. Manajemen & Konfigurasi
*   **Gemini API Key**: Aplikasi ini menggunakan Google Gemini AI. Secara default, API Key sudah dikonfigurasi secara aman di sisi server oleh pengembang. Anda tidak akan melihat kunci ini di kolom input demi keamanan. Jika Anda ingin menggunakan kunci pribadi, Anda dapat memasukkannya di sidebar, dan aplikasi akan memprioritaskan kunci Anda.
*   **Reset Semua Data**: Tombol untuk membersihkan seluruh riwayat jika Anda ingin memulai perjalanan hijau Anda dari nol lagi.
*   **Penyimpanan Lokal**: Semua data tersimpan aman di file `user_data.json` di perangkat Anda, memberikan kontrol penuh atas privasi data Anda.
